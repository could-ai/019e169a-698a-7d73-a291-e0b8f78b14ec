# 🌌 Квантовый прыжок к мечте (Quantum Leap)

Добро пожаловать в проект **«Квантовый прыжок к мечте»** — уникальную трансформационную игру-приложение, созданную специально для Всемирной ассоциации интеллект-тренеров SUPER JUMP. 

Миссия проекта: Мы создаём фильм-новеллу «Рай на Земле» через счастье и исполнение желаний каждого человека. В этой игре каждый участник выступает в роли актёра и режиссёра своей жизни, команда — это съёмочная группа, а жизненные ситуации — сцены для осознанного отыгрыша.

---

## 🎬 Обзор продукта

«Квантовый прыжок к мечте» — это интерактивная игра, которая проводит пользователя через 7 этапов трансформации. Она реализована как кросс-платформенное Flutter-приложение, которое идеально работает на мобильных устройствах, планшетах и в веб-браузерах.

### Основные фичи и 7 этапов игры:
1. **Кастинг и Барабан Сфер:** Игрок вращает колесо баланса (8 сфер жизни) и проходит квиз на определение ведущей энергии (Тиран, Жертва, Со-Творец). В результате выдаётся жанр восприятия жизни.
2. **Точка Б и Заявка:** Погружение в состояние мечты через сгенерированную нейросетью музыку и подача заявки в команду (на "кастинг").
3. **Командная игра (Алиас Эмоций):** Знакомство участников и интерактивная карточная игра с таймером на угадывание эмоций без слов.
4. **Образ «Я» в Точке Б:** Сборка нового портрета себя с помощью поддерживающих комплиментов от команды по специальной кино-формуле.
5. **Триумфальная речь:** Подготовка и (репетиция) записи финального монолога из состояния достигнутой цели.
6. **Удержание состояния (Парные этюды):** Интерактивные карточки с ситуациями (Тень и Дар). Перевод негативных реакций в осознанные решения с помощью практик SUPER JUMP.
7. **Действие из Нового «Я»:** Финальный шаг — выбор реального действия (пригласить друзей, помочь близкому, вступить в клуб). Выдача сертификата «Актёр фильма Жизнь мечты».

## 🚀 Как запустить проект

1. **Предварительные требования:**
   - Установите [Flutter SDK](https://docs.flutter.dev/get-started/install)
   - Установите IDE (VS Code или Android Studio) с плагинами Flutter/Dart.

2. **Сборка и запуск:**
   - Склонируйте репозиторий и перейдите в папку проекта.
   - Выполните установку зависимостей:
     ```bash
     flutter pub get
     ```
   - Для запуска веб-версии (рекомендуется для тестов):
     ```bash
     flutter run -d chrome
     ```
   - Для запуска на мобильном эмуляторе/устройстве:
     ```bash
     flutter run
     ```

3. **Сборка для продакшена (Web):**
   ```bash
   flutter build web --release
   ```
   Сгенерированные файлы будут находиться в папке `build/web/`. Их можно разместить на любом статическом хостинге (Firebase Hosting, GitHub Pages, Vercel и др.).
